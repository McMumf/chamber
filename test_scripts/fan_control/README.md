# Fan Control Tests

## Setup

1. Wiring
   1. Plug in the power to a 5v (red)
   2. Plug in Ground (black)
   3. Plugin in PWM (blue) to GPIO Pin 18
2. Execute the test script

    ```sh
    $ python controller.py
    Duty Cycle: 100
    Duty Cycle: 50
    Duty Cycle: 0
    Duty Cycle: 100
    Duty Cycle: 50
    Duty Cycle: 0
    Duty Cycle: 100
    ...
    ```
